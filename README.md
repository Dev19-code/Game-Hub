# 🎮 Game Hub (v1.0)

A polished, multi-game Flutter application featuring classic logic and reflex games. Built with a focus on clean architecture, state management, and immersive user experience.

---

## 🕹️ The Games

* **Hangman:** The classic word-guessing challenge. Don't let the stickman down!
* **Hot or Cold:** Use your intuition to find the secret number. Getting warmer?
* **Colored Words (Stroop Test):** A fast-paced cognitive challenge. Don't let the colors trick your brain!

## ✨ Key Features

* **Adaptive Theming:** Full support for **Dark Mode** and Light Mode via `Provider`.
* **High Score Persistence:** Personal bests are saved locally using `shared_preferences`.
* **Immersive Audio:** Custom sound effects for every game event (wins, losses, and interactions).
* **Modern UI:** Built with **Material 3**, featuring a dynamic dashboard and sleek list-based navigation.
* **Responsive Design:** Locked to portrait orientation for a consistent mobile experience.

## 🛠️ Tech Stack

* **Framework:** [Flutter](https://flutter.dev)
* **State Management:** [Provider](https://pub.dev/packages/provider)
* **Local Storage:** `shared_preferences`
* **Audio Engine:** `audioplayers`
* **Icons & Assets:** Custom game posters and localized sound effects.
